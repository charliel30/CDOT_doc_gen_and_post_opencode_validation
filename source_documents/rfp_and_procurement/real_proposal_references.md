# Publicly Available CM/GC Technical Proposals — Reference List

These are real, awarded technical proposals from state DOT CM/GC and design-build highway projects that are publicly hosted and freely accessible. They demonstrate actual proposal structure, language, level of detail, and winning approaches.

*Note: These PDFs were identified during research but not downloaded into this repository. They serve as reference for understanding what real proposals look like. The example proposals in this repo's `examples/` folder are modeled after these real-world documents.*

## Scoring Patterns Observed Across State DOTs

| State | Technical vs. Price Weight | Scoring Method |
|-------|---------------------------|----------------|
| CDOT (CM/GC) | Modified Satisficing (color: Green/Yellow/Red) | Qualitative with shortlist + interview |
| CDOT (Best Value) | 35% technical / 65% price | Point-based |
| FHWA (SEP-14 model) | 65% technical / 35% price | 100-point scale, 70-point minimum threshold |
| ARDOT | 1,000-point technical scale + 25 bonus points | Consensus committee scoring |
| Caltrans | 70% qualifications / 30% interview | Two-committee evaluation |
| Maryland SHA | Varies by project | Technical evaluated first, then price |
